📊 Academic Performance Analysis Dashboard

A professional multi-page Power BI dashboard designed to analyze academic performance at institutional, subject, and student levels.

This project demonstrates advanced data modeling, DAX calculations, performance ranking logic, and executive-style dashboard design.

🚀 Project Structure

The dashboard is divided into three analytical layers:

📄 Page 1 – Academic Performance Overview

Provides high-level institutional insights:

Total Students

Average SGPA

Highest SGPA

Pass Percentage

Total Subjects

Performance Category Distribution

College-wise SGPA Comparison

SGPA Distribution Analysis

📄 Page 2 – Deep Performance Analysis

Focused on subject-level and college-level performance insights:

Average Marks by Subject

Pass % by Subject

Subject Difficulty Index

College Performance Comparison

College Ranking

SGPA Trend Distribution

📄 Page 3 – Student Performance Drill Report

Detailed individual student analysis:

SGPA

Total Marks Obtained

Rank

Performance Category

Subject-wise Marks Breakdown

Internal vs External Marks Comparison

SGPA Performance Indicator

Strongest & Weakest Subject Identification
